What is HTML

HTML stands for HyperText Markup Language. HTML is not really a programming language. It is a markup language. Basically just controlling the formatting or structure of a text file.

What is CSS

CSS stands for Cascading Style Sheets. CSS is not really a programming language. It is a style sheet language. Basically it's used to control the presentation (like color, font or layout) of a markup language.
